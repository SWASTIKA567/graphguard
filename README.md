Overview
GraphGuardians is a cross-platform mobile application designed to bring graph-based vulnerability detection to your fingertips. It connects seamlessly to the GraphGuardians backend to provide real-time graph visualization, security vulnerability insights, and an intuitive interface for exploring complex node-edge relationships — all on mobile.
Whether you're a security researcher or a developer auditing dependencies, GraphGuardians gives you a powerful, portable tool to detect threats hidden in graph structures.
Feature | Description |

|  **Graph Visualization** | Interactive rendering of nodes and edges with zoom, pan, and tap-to-inspect |
|  **Real-time Data Display** | Live sync with the backend API — data updates reflected instantly |
|  **Backend API Integration** | Full REST integration with the GraphGuardians backend service |
|  **Vulnerability Detection UI** | Clear visual indicators for flagged nodes and risk levels |
|  **Node Exploration** | Tap any node to explore its connections, metadata, and risk score |
|  **Summary Dashboard** | At-a-glance stats: total nodes, vulnerable paths, threat level |
